The examples are provided by R. Cavoretto, A. De Rossi, S. Lancellotti, 
E. Perracchione. Cite as 
R. Cavoretto, A. De Rossi, E. Perracchione, S. Lancellotti, 
Software Implementation of the Partition of Unity Method, preprint

The routine MainPU contains examples concerning 
the computation of the PU interpolant. 

To run such codes the user needs the following Matlab functions.

G. E. Fasshauer, DistanceMatrix.m, available at:
http://www.math.usm.edu/cschen/COS702/DistanceMatrix.m

G. E. Fasshauer, MakeSDGrid.m, available at:
http://www.math.unipd.it/~demarchi/TAA2010/MakeSDGrid.m
